# Dashboard de Salários na Área de Dados

Dashboard interativo desenvolvido em **Python** utilizando **Streamlit** para análise de salários na área de Dados.

O projeto permite explorar dados salariais ao longo dos anos por meio de filtros dinâmicos, facilitando análises comparativas e exploratórias.

## Funcionalidades

- Filtros por **ano**, **senioridade**, **tipo de contrato** e **tamanho da empresa**
- Exibição de métricas principais:
  - Salário médio
  - Salário máximo
  - Total de registros
  - Cargo mais frequente
- Visualizações interativas com **Plotly**:
  - Top 10 cargos por salário médio
  - Distribuição de salários
  - Proporção dos tipos de trabalho (remoto, híbrido, presencial)
  - Mapa mundial com salário médio de Cientistas de Dados

## Tecnologias Utilizadas

- Python
- Streamlit
- Pandas
- Plotly

## Objetivo

Fornecer uma ferramenta visual e interativa para análise exploratória de dados salariais na área de Dados, auxiliando estudantes e profissionais na compreensão do mercado.
